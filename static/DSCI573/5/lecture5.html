Loss functions for regression

Ordinary least squares (OLS)

  • The loss function is \[J(w) = \sum_{i=1}^n (y_i - w^Tx_i)^2\]

  • Find \(w\) that minimizes the loss function.

  • The quadratic cost between the actual target \(y\) and the model predictions \(\hat{y}\).
  • Talk about matrix form.

Absolute value loss

  • If we do not want to punish large errors: \[J(w) = \sum_{i=1}^n \lvert y_i - w^Tx_i\lvert\]
  • Robust to outliers.
  • Plot the loss curve for the two different loss functions.

Scoring metrics vs loss functions

  • Scoring metrics:
    • Used for evaluation.
    • Used to find the best hyperparameters, e.g., the value of alpha.
  • Loss functions:
    • Used to optimization (find the best model parameters) during training.
    • Some flexibility to pick, but constrained by what is easy to optimize.
  • Scoring metrics:
    • We can pick those that are suitable for our business objective.
  • Loss functions:
    • Used to optimization (find the best model parameters) during training.
    • More internal to the model, e.g. the value of the coefficients.
    • Some flexibility to pick, but constrained by what is easy to optimize.
    • Squared errors are easy to optimize and common for regression (Ordinary Least Squares).
    • Can even solve squared errors analytically, so without the need for optimization, but in an ML context we will think of it
  • No MAPE etc, not as easy to optimize

  • The loss should be large when the model is very confident and wrong

0/1 loss

\[J(w) = \sum_{i=1}^n I(y_i w^Tx_i < 0)\]

  • We are giving the same penalty to the examples where we are only slightly incorrect vs very incorrect.
  • Draw the loss curve for 0-1 loss

Exponential loss

\[J(w) = \sum_{i=1}^n \exp(-y_iw^Tx_i)\]

  • It is encouraging larger values of \(y_iw^Tx_i\).
  • Draw the loss curve for exp loss

Hinge loss

\[J(w) = \sum_{i=1}^n \max(0, 1 - y_iw^Tx_i)\]

  • Confident and correct examples are not penalized.

Logistic loss

\[J(w) = \sum_{i=1}^n \log\left(1+\exp({-y_iw^Tx_i})\right)\]

Binary cross-entropy loss

  • When the class labels are 0 and 1, we have the cross-entropy loss: \[f(w) = \sum_{i=1}^n -y_i \log(\hat{p}_i) - (1-y_i) \log(1-\hat{p}_i)\]
  • Logistic loss and binary cross-entropy loss are equivalent (your exercise).

Which loss function should I use?

  • 0/1 is not easy to optimize.
  • Exp loss is not robust to outliers.
  • Hinge loss is used in SVMs.
  • Logistic loss is used in logistic regression.
  • Cross-entropy loss is widely used in neural networks.

Loss function for multi-class classification

  • Cross-entropy loss extends naturally to multi-class classification:

\[f(w) = \sum_{i=1}^n \sum_{c=1}^C -y_{i,c} \log(\hat{p}_{i,c}).\]

Regularization

Model complexity

  • Different degrees of polynomials.
  • What did you see?
    • Training score goes up.
    • Validation score goes up then down.
  • When we increase the degree, do we increase the model complexity or decrease it?

How do you quantify complexity?

  • Number of features.
  • The size of the parameter space (e.g., the magnitude of the coefficients/model parameters).

Regularization

  • Complex models tend to overfit.
  • Minimize the loss function + complexity penalty.
  • L0, L1, L2 norms on the model parameters.
  • Adds a complexity penalty multiple by lambda to the loss function

L0 L1 L2 Norms

Norms

We want to measure the size of a vector!

  • L0: \(\lVert w \rVert_0\) = The number of non-zero element
  • L1: \(\lVert w \rVert_1 = \lvert w_1 \rvert + \lvert w_2 \rvert + \dots + \lvert w_n \rvert\)
  • L2: \(\lVert w \rVert_2 = (w_1^2 + w_2^2 + \dots + w_n^2)^{1/2}\)
  • L1: The sum of the absolute values
  • L2: The square root of the sum of the squared values
